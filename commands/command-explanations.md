# Command Explanations

This document provides a brief explanation of the command categories used during this incident response case study.

| Category         | Purpose                                      |
| ---------------- | -------------------------------------------- |
| Nmap             | Network reconnaissance and service discovery |
| Hydra            | SSH brute-force attack simulation            |
| SSH              | Remote access to the target system           |
| User Enumeration | Collect user and system information          |
| grep             | Search authentication logs                   |
| awk              | Extract specific fields from log entries     |
| sort             | Sort extracted data                          |
| uniq             | Remove duplicates or count repeated values   |
| wc               | Count matching log entries                   |
| head             | Display the first event                      |
| tail             | Display the last event                       |
| last             | Review login history                         |
| history          | Display executed commands                    |
| systemctl        | Manage the SSH service                       |
| truncate         | Clear authentication logs                    |
| touch            | Create evidence files                        |
| cat              | Display file contents                        |

These commands collectively demonstrate the complete workflow of preparing the environment, simulating an attack, investigating the incident, collecting evidence, and documenting the findings.
