# Evidence Summary

This directory contains the digital evidence collected during the investigation of the simulated SSH brute-force attack.

## Evidence Collected

* Authentication log analysis
* System log analysis
* Attacker IP address
* Failed login attempts
* Successful login events
* SSH session records
* User activity
* Account information
* Incident timeline

The collected evidence confirms that the attack originated from a single IP address and successfully compromised a standard user account without obtaining administrative privileges.

All evidence was collected from a controlled laboratory environment and preserved for educational purposes.
